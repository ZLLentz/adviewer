adviewer
===============================

.. image:: https://img.shields.io/pypi/v/adviewer.svg

AreaDetector configurator and viewer

Requirements
------------

* Python 3.6+
* qtpy, PyQt5
* qtpynodeeditor

Running adviewer
----------------

```sh
$ pip install git+https://github.com/pcdshub/adviewer
$ adviewer --help
$ adviewer 13SIM1:
$ adviewer --pvlist filename.pvlist
```

Running the Tests
-----------------
::

  $ python run_tests.py
